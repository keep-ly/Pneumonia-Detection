## 项目结构

```
专业综合训练/
├── data/                          # 数据集目录（待放入 chest_xray）
├── configs/
│   └── config.yaml                # 超参数配置文件
├── src/
│   ├── utils.py                   # 随机种子、早停、类别权重、指标计算
│   ├── data_loader.py             # 数据加载与增强（ImageFolder + transforms）
│   ├── model.py                   # 模型定义（ResNet50/EfficientNet-B0/自定义CNN）
│   ├── train.py                   # 训练主程序（混合精度 + TensorBoard + 早停）
│   └── evaluate.py                # 评估脚本（ROC、混淆矩阵、指标保存）
├── outputs/
│   ├── checkpoints/               # 模型权重保存
│   ├── logs/                      # TensorBoard 日志
│   └── results/                   # 评估图表输出
└── requirements.txt               # Python 依赖
```

## 核心实现要点

| 模块 | 关键特性 |
|------|---------|
| `data_loader.py` | ImageFolder 自动按目录读取标签，训练集做水平翻转/旋转/亮度增强，验证测试集仅归一化 |
| `model.py` | 支持 ResNet50（默认）、EfficientNet-B0、自定义4层CNN，预训练模型替换全连接层为 Dropout+Linear(1) |
| `train.py` | 加权 BCEWithLogitsLoss、Adam + CosineAnnealing、CUDA 混合精度、EarlyStopping(patience=7)、TensorBoard 记录 |
| `evaluate.py` | 输出 Accuracy/Precision/Recall/F1/AUC-ROC/混淆矩阵，保存 ROC 曲线图和混淆矩阵热力图 |
| `utils.py` | set_seed、自动计算 pos_weight、EarlyStopping 类、sklearn 指标封装 |

## 使用方法

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 下载数据集并解压到 data/chest_xray/
#    下载地址: https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia

# 3. 训练
python src/train.py

# 4. 查看训练曲线
tensorboard --logdir outputs/logs

# 5. 评估
python src/evaluate.py --checkpoint outputs/checkpoints/best_model.pth
```

配置参数集中在 [configs/config.yaml](file:///c:/Users/HW/Desktop/专业综合训练/configs/config.yaml) 中，可根据需要调整 `batch_size`、`epochs`、`learning_rate` 等。
